//INSTALL
  enable the module :)
  Create a content type or edit an existing one
  Add a field that allows a multiselect like a node reference field for example and select the fancy multiple select in the widget field. (Dont forget to set your instance to multiple)

//DEPENDENCIES
  This module depends on optionwidgets
  This module depends on the jquery_multiselect module (which depends on jquery_update and jquery_ui see that modules README for instructions)
