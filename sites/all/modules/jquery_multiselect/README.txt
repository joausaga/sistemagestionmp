//DEPENDENCIES
  
  http://drupal.org/project/jquery_ui with the 1.7 library and the 1.7 lib requires http://drupal.org/project/jquery_update

//INSTALL
  
  Download the jquery multiselect plugin on http://github.com/michael/multiselect/zipball/master
  Extract the contents of the folder into /jquery_multiselect/jquery
  Enable module
  To include the plugin on a page call jquery_multiselect_plugin_add()

//INTEGRATION
  
  This module integrates with http://drupl.org/project/jq

//OTHER MODULES THAT USE THIS MODULE
  
  This module is needed by the fancy multiselect form element wich can make any multiselect in drupal use the jquery ui multiselect
