-- SUMMARY --

The user_access module stores informations about user and node each time a node
is viewed. These informations may be then used in order to built site usage
report.

For a full description of the module, visit the project page:
  http://drupal.org/project/user_access

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/user_access


-- REQUIREMENTS --

None.

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.
